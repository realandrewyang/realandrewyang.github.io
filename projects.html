Nifty Stock Tool

Although still under construction, the Nifty Stock Tool will be a clean, easy-to-use stock tool for the everyday investor once it is completed. It will provide real-time analysis of stocks trading on the NASDAQ, NYSE, and TSX.

RPN Calculator

For my Grade 12 Computer Science Class, I created an RPN Calculator in C++ using object-oriented principles. I also used data structures such as stacks and queues (implemented as linked lists).

UNI-te

Created at Hack the North 2018, UNI-te is a Google Home voice application that helps students plan their time. What makes UNI-te special is that it's integrated with the student's school's suite of websites (i.e. LEARN, CrowdMark, etc.)

Online Store API

I created this RESTful API with Node.js and MongoDB to serve as a generic backend for a simple online store. Nearly complete, this API supports inventory queries, full list queries, and purchase queries in the context of a shopping cart.
