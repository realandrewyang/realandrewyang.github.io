Nifty Stock Tool

Although still under construction, the Nifty Stock Tool will be a clean, easy-to-use stock tool for the everyday investor once it is completed. It will provide real-time analysis of stocks trading on the NASDAQ, NYSE, and TSX.

The Toronto Stock Evaluator

The precursor to the Nifty Stock Tool, this prototype website displays real-time stock information and provides recommendations for buying or selling individual tickers.

UNI-te

Created at Hack the North 2018, UNI-te is a Google Home voice application that helps students plan their time. What makes UNI-te special is that it's integrated with the student's school's suite of websites (i.e. LEARN, CrowdMark, etc.)

hs_math

hs_math is a Python library I created when I first started learning programming. Features include methods for deriving linear equations, computing permutations, combinations, and prime factorization, among other functions that are useful for high school math students.
